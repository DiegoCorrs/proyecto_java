Requerimientos del Proyecto

Requerimientos Funcionales
1. El programa debe permitir al usuario ingresar los siguientes datos de un vehículo:
    - Marca
    - Modelo
    - Cilindrada
    - Tipo de Combustible
    - Capacidad en Pasajeros
2. El programa debe permitir al usuario ingresar el monto total de la compra.
3. El programa debe permitir al usuario ingresar la distancia al destino en kilómetros.
4. El programa debe calcular el costo de envío siguiendo las siguientes reglas de negocio:
    - Si el monto total de la compra es mayor a $50,000 o la distancia al destino es menor o igual a 20 km, el envío es gratis.
    - Si el monto total de la compra está entre $25,000 y $49,999, se cobrará $150 pesos por kilómetro recorrido.
    - Si el monto total de la compra es menor a $25,000, se cobrará $300 pesos por kilómetro recorrido.

Requerimientos No Funcionales
1. El programa debe estar desarrollado en Java.
2. El programa debe ser compilado utilizando el compilador de Java (javac) desde la línea de comandos.
3. El programa debe ser ejecutable en cualquier entorno de Java compatible.
4. El código fuente del programa debe ser documentado adecuadamente, incluyendo comentarios claros y concisos en cada línea de código.
5. La documentación completa del proceso, incluyendo el código fuente y el seguimiento detallado, debe ser almacenada y gestionada utilizando GitHub.
6. El programa debe solicitar y mostrar los datos al usuario de manera clara y comprensible.
7. El programa debe cumplir con las reglas de negocio establecidas para el cálculo del costo de envío.
